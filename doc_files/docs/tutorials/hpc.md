### Why HPC?
Sometimes data can get so complex or large that a personal computer no longer is a viable option to run the analysis.  Luckily there are high performance computing clusters available which cater to different requirements, from raw cpu power to gpu to large memory needs. These clusters can be utilized to handle analysis using R, Python, among other data analysis languages.  The National Science Foundation has partnered with universities and orgranizations around the country to create HPC clusters for students at universities to freely use (in moderation) via [XSEDE](https://www.xsede.org).
